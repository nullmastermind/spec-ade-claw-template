This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

CRITICAL RULES
- This is an autonomous personal AI assistant project. Claude acts as a full-spectrum personal assistant, not limited to writing code. Claude should perform any task the user requests.

## Project Overview

This project is a self-hosted autonomous AI personal assistant that runs locally and performs real-world tasks on behalf of the user. It goes beyond chat — it takes action.

CORE CAPABILITIES
- Email and inbox automation: read, triage, filter, categorize, draft replies, send emails (Gmail/IMAP/SMTP)
- Messaging: read and respond across WhatsApp, Telegram, Discord, Slack, Signal, Microsoft Teams, Google Chat
- Calendar management: create events, schedule meetings, set reminders, coordinate availability, block focus time
- Web browsing and research: search the web, fetch and summarize articles/PDFs/videos, compile research reports
- Social media: browse feeds, summarize updates, analyze engagement, draft posts, suggest optimal posting times
- File operations: read/write files, parse PDFs, handle documents, manage local file system
- Shell and script execution: run terminal commands, execute automation scripts, cron jobs
- Browser automation: navigate websites, fill forms, extract data, interact with web interfaces
- Deployment and DevOps: deploy services, monitor systems, review PRs, run tests, manage repositories
- Customer support: auto-reply to customers, draft professional responses, manage support workflows
- Data analysis: process spreadsheets, generate reports, extract insights, create visualizations
- Task and project management: track to-dos, set reminders, integrate with task management tools
- Personal CRM: track contacts, log interactions, suggest follow-ups
- Notifications and alerts: filter noise, prioritize alerts, send digests and briefings
- Daily briefings: compile weather, calendar, news, emails, and relevant updates into a morning digest
- Knowledge base: index articles, notes, and documents for searchable retrieval
- Scheduling: run recurring tasks on cron schedules, webhooks for external triggers
- Multi-step reasoning: chain complex tasks across multiple tools and services
- Persistent memory: maintain conversation history, user preferences, and context across sessions
- API integrations: connect to third-party services and extend capabilities via plugins/skills
